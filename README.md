## 🚀 核心功能

- ✅ 小说聚合首页【独家支持】可嵌入Flutter 页面
- ✅ 小说阅读页面
  - 4 种翻页效果
  - 黑暗模式
  - 字体大小调节
  - 亮度调节
  - 护眼模式
  - 背景色更换
- ✅ 数据接口
  - 获取类目列表
  - 获取短剧列表根据类目
  - 获取推荐短剧列表
  - 获取短剧详情根据 id
  - 搜索短剧
  - 收藏/取消收藏短故事
  - 获取收藏列表
  - 获取历史阅读记录
